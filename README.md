# CleanWalk – AI-Based Cleanliness Reporting App

CleanWalk is a mobile application developed using Flutter (Dart) that allows users to report cleanliness issues using image and location input.

## Features
- AI-based image classification (TensorFlow Lite)
- Map-based location selection
- Real-time reporting using Firebase
- NGO event management system
- Volunteer participation

## Technologies Used
- Flutter (Dart)
- Python (AI model training)
- TensorFlow Lite (AI integration)
- Firebase (Firestore, Authentication, Storage)

## Purpose
This project aims to improve urban cleanliness by enabling users to report issues and allowing NGOs to manage cleanup activities efficiently.
